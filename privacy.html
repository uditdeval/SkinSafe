Privacy and transparency

Privacy Policy

SkinSafe values user trust and keeps this website simple, safe, and transparent for educational use.

Information Usage

SkinSafe does not store personal skincare inputs permanently. The ingredient information entered by the user is used only to generate on-screen analysis results within the current browsing session.

No Login Required

Users can explore the website without creating any account or profile.

02

No Permanent Storage

Ingredient details entered on the site are not saved in a database.

03

Educational Use

This website is designed for demonstration and learning purposes only.
